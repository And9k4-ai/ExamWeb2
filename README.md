📂 Cara Menjalankan Aplikasi di Lokal

Karena project ini murni dibangun menggunakan HTML, CSS, 
dan JavaScript statis, Anda tidak membutuhkan langkah kompilasi (*build process*).
                                                                                                                                                                        
1. *Clone* repositori ini ke komputer Anda atau unduh berkas ZIP-nya.
2. Buka folder hasil download.
3. Klik dua kali pada file `index.html` untuk membukanya secara langsung 
di browser favorit Anda (Chrome, Edge, Firefox, atau Safari).
